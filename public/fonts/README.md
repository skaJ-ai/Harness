# Samsung One 폰트 설치

이 디렉토리에 Samsung One 폰트 파일(.woff2)을 넣어주세요.

## 필요한 파일

```
fonts/
├── SamsungOne-400.woff2    (Regular)
├── SamsungOne-500.woff2    (Medium)
├── SamsungOne-700.woff2    (Bold)
├── SamsungOneKR-400.woff2  (Korean Regular)
├── SamsungOneKR-600.woff2  (Korean SemiBold)
└── SamsungOneKR-700.woff2  (Korean Bold)
```

## 다운로드 경로

- 사내: https://design.samsung.com/kr/contents/samsungone/
- 또는 samsung.com 소스에서 woff2 파일 추출 가능

## 폰트 없이 실행 시

시스템 폰트(Apple SD Gothic Neo, Malgun Gothic 등)로 자동 폴백됩니다.
