# Pretendard Variable 폰트 설치

이 디렉토리에 Pretendard Variable 폰트 파일을 넣어주세요.

## 필요한 파일

```
fonts/
└── PretendardVariable.woff2    (Variable Font, 45-920 weight)
```

## 다운로드

- GitHub: https://github.com/orioncactus/pretendard/releases
- CDN (개발용): https://cdn.jsdelivr.net/gh/orioncactus/pretendard@v1.3.9/dist/web/variable/pretendardvariable.min.css

## 폰트 없이 실행 시

시스템 폰트(-apple-system, Segoe UI 등)로 자동 폴백됩니다.
