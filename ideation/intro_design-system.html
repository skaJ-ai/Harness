User Journey
이렇게 동작합니다
아이디어 한 줄이면 시작됩니다. AI가 질문하고, 정리하고, 만들어줍니다.
1

아이디어를 입력합니다

"신입사원 온보딩 프로세스를 개선하고 싶어" — 이 한 줄이면 충분합니다.

자유 텍스트한 줄도 OK
2

AI가 인터뷰합니다

"목적이 뭔가요?", "대상은 누구인가요?", "현재 상황은?" — AI가 기획에 필요한 7가지를 물어봅니다.

목적대상현황제안기대효과일정산출물
3

기획서가 자동 생성됩니다

인터뷰 내용을 바탕으로 구조화된 기획서가 만들어집니다. 수정도 자유롭게.

배경현황 분석제안 사항실행 계획
4

원하는 형태로 변환합니다

같은 기획서에서 보고서, PPT, 엑셀이 한 번에 나옵니다. 톤도 선택 가능합니다.

보고서PPT 슬라이드엑셀경영진용실무용발표용
5

AI가 한 번 더 검토합니다

"근거가 약합니다", "숫자를 추가하세요" — 팀장/임원 관점에서 피드백을 줍니다.

Features

💬

AI 인터뷰 채팅

아이디어를 입력하면 AI가 구조화된 질문으로 기획을 도와줍니다.

📑

자동 기획서 생성

배경, 현황, 제안, 기대효과가 포함된 기획서가 자동으로 만들어집니다.

🎯

톤 변환

같은 기획서를 경영진/실무/발표용 3가지 톤으로 자동 변환합니다.

📊

PPT / 엑셀 / 보고서

기획서에서 슬라이드, 차트 엑셀, 구조 보고서를 원클릭 생성합니다.

✅

AI 리뷰어

올리기만 하면 "근거 약함", "ROI 추가하세요" 등 즉시 피드백.

🔗

문서 지식 그래프

문서 간 관계를 자동 추적. 관련 자료를 한눈에 파악합니다.

📂

문서 임포트

기존 Word, PPT를 업로드하면 자동 변환되어 DB에 저장됩니다.

🔍

전문 검색

모든 문서에서 키워드 검색. 과거 자료를 즉시 찾습니다.

🔄

반복 보고서

주간/월간 보고 템플릿에 변경점만 입력하면 자동 완성.

Knowledge Graph
문서가 연결되면, AI가 맥락을 압니다
단순 검색이 아닙니다. 문서 간 관계를 추적해서 AI가 더 정확한 답을 내놓습니다.
"온보딩 개선 기획서를 써줘" → AI가 관련된 모든 문서의 맥락을 파악하고 반영합니다.

AI-Native Strategy
AI Native 업무 환경 전환의 Phase 1
"AI Agent를 만드는 게 아니라, AI Agent가 제대로 일할 수 있는 환경을 만드는 것."
현재 문제
로컬 기반 퇴직 시 업무 지식 소실
서식 노이즈 AI가 읽을 수 없는 문서
데이터 사일로 AI Agent 도입 기반 부재

로컬에 있는 데이터는 파일이고, 연결된 데이터는 자산입니다.
